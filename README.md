# Boutique FoodKwetu

**Live Website:** [https://foodkwetu.vercel.app](https://foodkwetu.vercel.app)

FoodKwetu is an e-commerce platform for African food products, serving both individual customers and professionals.

## Tech Stack

- [Next.js 14](https://nextjs.org/) — React framework
- [Tailwind CSS](https://tailwindcss.com/) — Styling
- [Supabase](https://supabase.com/) — Database & backend
- [Zustand](https://zustand-demo.pmnd.rs/) — State management
- [Vercel](https://vercel.com/) — Deployment

## Pages

- `/` — Homepage
- `/produits` — Product catalogue
- `/professionnels` — Professional/B2B section
- `/admin` — Admin dashboard

## Getting Started

1. Clone the repository
2. Copy `.env.local.example` to `.env.local` and fill in your Supabase credentials
3. Run `npm install`
4. Run `npm run dev`
